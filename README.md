# medik
Manage your Windows apps from command line. Useful for automation or if you afraid GUI.

## Usage
#### Update an app
```bash
medik update --id="APP_ID"
```

#### Print app list
```bash
medik list
```

#### Restore medik to defaults
```bash
medik reset
```

#### Help
```bash
medik help
```

## Install
```bash
sudo npm install medik -g
```

## Build
```bash
npm install
gulp
```
